# CDN Diagnostic Tool 
A website diagnostic tool built with Node.js, Express and vanilla JavaScript. The tool gathers and displays the following information about websites:
 - DNS records
 - CDN provider
 - Cachability
 - TLS/SSL certificate
 - HTTP protocol version
 - Time To First Byte
 # Installation and Setup
    git clone <repo URL>
    cd cdn_tool
    npm install
    node server.js
